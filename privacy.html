Privacy Policy — herox

Last updated: October 2025

Introduction
herox is a Discord bot that helps communities with moderation, ranking, activity tracking, and automated server utilities. This Privacy Policy explains what data the Bot collects, how it is used, and how users can control it.

Data we collect

The Bot may collect and temporarily process the following data within Discord servers:

  • User IDs and usernames (for identification and ranking)
  • Message content when processing commands (e.g. !rank, !verify)
  • Presence and activity status (to calculate online time or activity rewards)
  • Server and channel IDs (for configuration and logging)

No personal data is collected outside of Discord unless explicitly stated.

How we use the data

Data is used solely to:

  • Respond to user commands and manage moderation features
  • Track member activity for XP or rank systems
  • Provide activity logs and server utilities to administrators

Data storage

The Bot does not store message content or presence data outside of Discord. Basic user statistics (such as XP, ranks, or configuration) may be stored securely in the Bot's database for feature functionality. No user data is shared with third parties.

User control & opt-out

Users and server administrators can:

  • Disable rank/activity tracking via bot commands or server configuration.
  • Request deletion of stored data by contacting the bot developer or using the /data-delete command if available.

Data security

Reasonable measures are taken to protect stored data against unauthorized access. Only the bot owner and authorized maintainers have access to the bot's database.
